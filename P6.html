Problem 6

Given x and n as input, find the sum of the following series

Constraints

1 <= T <= 10
1 <= x <= 100
0 <= n <= 100

Input Format

The first line contains the number of test cases, T. Then T lines follow each containing the values of x and n separated by a space.

Output Format

Print the sum of the given series up to four decimal places.

Sample Input

2
5 0
2 1

Sample Output

1.0000
0.5000
