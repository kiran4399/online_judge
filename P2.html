Problem 2

Given a set of numbers, write a program to print the minimum of those numbers. First input is n the number of values, and after that you have n numbers.

Constraints

-10^18 <= a[n] <= +10^18
0 <= n <= 10000000

Sample Input

8
7 5 6 2 3 7 9 8

Sample Output

2
