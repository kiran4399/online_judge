Problem 1

Given an integer n, print all the integers in reverse order starting from n upto 0, after corresponding number of spaces. First line denotes the number of test cases t.

Constraints

0 < t < 1000
0 <= n <= 1000

Sample Input

1
5

Sample Output

     5
    4
   3
  2
 1
0
