Problem-1

Given a two dimensional matrix A n*n , where every entry in the matrix is either 0 or 1. Initially Mr.X is at position (0,0). He would like to go to the position (n - 1, n - 1). If he is at A[i][j], then you can move one step down
(i.e., A[i + 1][j] ) or one step right (i.e, A[i][j + 1] provided if their values are 1. Mr.X would like to find out whether there is a way to reach his target or not. Write a C++ program to help Mr.X.
Relaxations: You can assume that the number of rows and columns in the input matrix are always 10.
Constriants: Use stacks.
Input format: The first line contains an integer m that indicates the number of test cases, followed by m * 100 binary values.
Output format: Print yes or no in individual lines for all the test cases
