Problem 4(BONUS QUESTION)

You are given n sticks, each of a given length. You need to create a (closed) polygon with as many of these sticks as possible. The polygon area should be greater than zero. As usual number of test cases are first.
Ex: If you have sticks of size 1, 5, 5, and 20. You cannot form a polygon using stick of size 20, as the sum of all the other sticks together is less than 20, but we can create a polygon with sizes 1, 5, and 5. So the answer is 3.

Constraints

0 < t <= 10
0 <= n <= 2000

Sample Input

1
4
1 5 5 20

Sample Output

3
