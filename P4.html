Problem 4

Given two matrices A and B, find A*B (Multiply two matrices).

Constraints

1 <= T <= 10
1 <= each dimension <= 100
-1000 <= each number in the matrix <= 1000

Input Format

The first line will be the number of test cases, T. For each test case, the dimension of the first matrix will be given followed by the first matrix, then the dimensions of the second matrix followed by the second matrix.

Output Format

Print the resultant matrix (after multiplication) with each row in a new line.

Sample Input

1
2 2
2 2
1 2
2 1
2
1

Sample Output

6
4
