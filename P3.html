Problem 3

Write a program to take a string, convert all the alphabets in the whole string to uppercase and print the string. Use EOF to check if you reached the end of input or not. (Note: [^D] means EOF). If any other character is there in the string then do not change it. First line takes the number of test cases n. Then Follows n strings.

Constraints

0 < len(string) <=300
0 <= n <= 10

Sample Input

1
IiitS[^D]

Sample Output

IIITS
