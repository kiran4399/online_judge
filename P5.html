Problem 5

Given a maze of '. 's and '#'s and a source and a destination, find if there exists a path from the source to the destination. You can travel only through the '.'s.
You cannot move diagonally.
If a path exists, print YES, Otherwise, print NO . The coordinates of the source and destination will be the respective row and column numbers.

Constraints

1 <= T <= 10
1 <= m, n <= 100
A(i,j) belongs to {'.','#'}
0 <= x1, x2, y1, y2 <=99

Input Format

The first line contains the number of test cases, T. For each test case, you will be given the dimensions, mXn, of the maze, followed by the contents, A(i,j), of the maze and then the coordinates, x1, y1, x2, y2, of the source and the destination.

Output Format

Print YES if a path exists and NO if no path exists (without quotes).

Sample Input

2
5 4
.  .  # .
.  .  . #
# # .  .
# .  .  .
.  .  # .
0 0
4 0
3 3
# . #
.  . #
. # .
0 1
2 2

Sample Output

YES
NO
